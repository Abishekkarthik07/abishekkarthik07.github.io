## GeoExpert-QA

GeoExpert-QA is a zero-shot geospatial question-answering agent capable of complex spatial reasoning, code generation, and dynamic map visualization. Built with Python, PyTorch, GeoPandas, Folium, and Streamlit, the system uses a multi-expert architecture to answer rich geographic queries and generate executable analysis code.

### Key Features
- Zero-shot geospatial reasoning
- Executable Python code generation
- Dynamic map visualization and GeoPatch outputs
