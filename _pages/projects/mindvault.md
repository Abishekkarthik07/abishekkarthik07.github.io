## MindVault – AI Personal Knowledge Chatbot

MindVault is an AI-powered personal knowledge assistant for uploading and querying documents via vector search and LLMs. It uses LangChain for text extraction and chunking, FAISS for semantic search, and large open models for contextual answers.

### Tech Stack
Python, Streamlit, LangChain, FAISS, HuggingFace, Groq API
